Angular 101

NOTE: This is a review of ANGULAR, we do not care about DESIGN.

How to start a project?

npm install -g @angular/cli

In my case, /usr/bin/ng file already exists in my system for some reason. I just made a back up of that and deleted it.

sudo cp /usr/bin/ng /usr/bin/ng.bk
sudo rm /usr/bin/ng
npm install -g @angular/cli

Create a project

ng new angular-tutorial

Say "no routing"

Would you like to add Angular routing? (y/N)
